# Features Included:
✅ Real-time hand tracking using MediaPipe
✅ Finger counting with visual feedback
✅ Gesture recognition (Fist, Point, Peace, Three, Four, Five, OK, Thumbs Up/Down, Victory)
✅ Multiple hand support (up to 2 hands)
✅ FPS counter and performance monitoring
✅ Bottom finger counter display
✅ Web interface with HTML/CSS/JavaScript
✅ Screenshot functionality
✅ Left/Right hand detection
✅ Stable gesture detection
✅ Customizable configuration
